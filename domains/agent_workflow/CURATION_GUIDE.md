# Agent Workflow Curation Guide

## 高价值记录

- 可直接复制给其他智能体的工作流提示词。
- 本地 API 调用约定。
- Windows 路径、PowerShell、Bash 兼容规则。
- 任务执行前的检索策略。
- 完成后的证据报告格式。
- 常见失败原因和恢复步骤。

## 低价值记录

- 没有执行步骤的泛泛建议。
- 只描述工具名称但没有调用方式。
- 不能验证的经验判断。
- 把所有知识一次性塞给模型的流程。

## 单条记录应回答

- 这个流程解决什么问题？
- 哪类智能体适用？
- 输入是什么？
- 需要调用哪些 API 或脚本？
- 失败时怎么恢复？
- 最终报告必须包含什么证据？
