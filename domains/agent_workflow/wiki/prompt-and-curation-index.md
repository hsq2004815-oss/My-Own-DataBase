# Prompt and Curation Index

## Purpose

This index connects reusable prompts, curation guides, ingestion prompts, and Codex workflow notes that help agents maintain this database.

Use this index when the task is about:

- importing or curating new source material
- turning raw material into rules, patterns, templates, or topics
- designing prompts for Codex, Claude Code, or opencode
- maintaining database quality without polluting raw sources
