# Landing Page Section Patterns

Premium landing pages need varied section rhythm, not a stack of identical cards.

## Useful Sections

- Hero.
- Logo Cloud / Partners.
- How It Works.
- Capabilities.
- Feature Chess.
- Feature Grid.
- Stats.
- Testimonials.
- CTA Footer.
- Portfolio Bento.
- Journal.
- Parallax Gallery.

## Hero Priority

For Landing Page, SaaS Homepage, Portfolio First Screen, and product launch
tasks, choose the hero composition before building later sections. Use
`video-hero-layout-patterns.md` together with `cinematic-video-hero-rules.md`.

Hero categories:

- Campaign Hero: top-aligned, uppercase giant title, custom SVG CTA, strong
  visual background.
- Cinematic Poster Hero: bottom-left content, full-screen video, directional
  overlay, editorial typography.
- SaaS App Hero: centered content, tagline pill, dual CTA, transparent
  navigation, mobile full-screen menu.
- Creative AI Split Hero: two-panel Liquid Glass layout, left narrative, right
  information cluster, video background.
- Technical Education Hero: HLS video, technical grid, SVG glow, floating glass
  info card, career-ready eyebrow, uppercase headline, single accent
  punctuation.
- Strict Grayscale Hero: no colored accent; use glass, grayscale opacity,
  typography, video, and spacing.

Do not use the Web App / Dashboard `liquid-glass-web-app-ui-kit.md` as the
default for marketing hero pages. It belongs to persistent app surfaces.

## Feature Rows

- Alternate text and media.
- Use large headings.
- Keep body copy short.
- Media container uses glass, rounded mask, or video/image background.
- Avoid generic screenshots in tiny cards.

## Bento Grid

- Do not make all cards the same size.
- Use 7/5 and 5/7 columns.
- Mix large and small cards.
- Use image/video background with hover overlay.
- Keep card count restrained.

## Stats

- Large number.
- Small label.
- Use serif italic or tight sans for premium contrast.
- Avoid table-like presentation.

## Testimonials

- Quote: muted white, optionally italic.
- Name: strong white.
- Role: `white/50`.
- Card: liquid-glass, `rounded-2xl`, `p-8`.

## Logo Cloud

- Use white/inverted logos on dark background.
- Can use infinite horizontal slider.
- Keep opacity controlled.
- Do not overload with too many logos.

## CTA Footer

- Return to the dominant visual direction.
- Use one primary CTA.
- Keep copy short and memorable.
- Use a final video/image/glass field only if it reinforces the page story.
