This index connects curated UI design reference nodes to the `ui_design` knowledge network.

Use this as a reference layer after reading the premium UI rules. Brand references are flavor and evidence, not the default rule layer.

## How to Use

- Read premium rules first.
- Use brand references to choose visual flavor, density, rhythm, and interaction feel.
- Prefer `wiki/topics` and `wiki/summaries` for task context.
- Use `processed/cleaned_text` only for provenance or curation review.
- Do not read `raw/` design source by default.
